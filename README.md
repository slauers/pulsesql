# Blacktable

Blacktable is a lightweight SQL workstation focused on speed, clarity, and flow.

Built for developers who are tired of slow, bloated database tools.

---

## Features

- Multi-connection workflow
- Query tabs with fast execution
- Result grid with filtering
- Metadata explorer (schemas, tables)
- Connection logs with visibility controls
- Table autocomplete (phase 1)
- PostgreSQL and Oracle support
- Semantic UI feedback (optional)

## Philosophy

Blacktable is designed around a few simple principles:

- Fast over feature-heavy
- Clear over clever
- Focused over everything-at-once

It prioritizes real workflows:

- Writing queries
- Inspecting data
- Switching contexts quickly
